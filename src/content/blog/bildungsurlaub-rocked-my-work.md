I signed up to learn. I did not expect to ship.

Five days, remote. A [Generative KI mit Python](https://www.bildungsurlaub.de/bildungsurlaub-angebote/generative-ki-mit-python/135837) Bildungsurlaub — Python, RAG pipelines, LangChain agents, vector databases, local LLMs. I wanted to understand how this stuff actually worked under the hood.

On day four, while the instructor was covering RAG, I opened a new project folder and started building a prototype I'd been circling for months. I hadn't planned to build anything that week. It just became obvious that I could.

---

## At work

I stopped waiting for engineering capacity. I built a performance analysis tool that now guides decision-making for customer success — without waiting for an engineering ticket.

I think differently about roadmaps — because I can now feel the difference between what's genuinely hard, what's trivially automatable, and where AI changes the complexity curve entirely.

My default laptop mode shifted too. Jira, Google Docs, and Slack used to be the whole screen. Now it's terminals — inside Project IDX, Positron, Obsidian, VS Code. The tools changed because what I'm doing changed.

## Personally

The projects I would have abandoned half-finished before now get shipped. The [Robot Cooks Drawings](/blog/robot-cooks-drawings) app. The [tax automation](/blog/tax-flow-claude-code). The [Claude Code build process](/blog/pm-workflow-ai-tickets). The [campaign analysis](/blog/campaign-analysis-automation). None of those felt completable before the course. Now they're live.

---

The week didn't teach me to code. It taught me what I was actually capable of building. That turns out to be a much bigger shift.
